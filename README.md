## About This Repository
### Coderminal CLI
Coderminal CLI is a modern vibe-coding application which runs direclty on your **Command Prompt**, **Powershell** or **Terminal**.
- **Written In**: Rust
- **Status**: Not developed yet

To know more about Coderminal CLI, [CLICK HERE](cli/README.md).

### Coderminal VScode Extension
Coderminal VScode Extension is a modern vibe-coding application which runs direclty in your VS code.
- **Written In**: TypeScript
- **Status**: Not developed yet

To know more about Coderminal VScode Extension, [CLICK HERE](cli/README.md).

### Coderminal Web
Coderminal Web is a modern vibe-coding application which runs direclty on your browser using the address [app.coderminal.com](app.coderminal.com).
- **Written In**: TypeScript
- **Status**: Not developed yet

**NOTE:** Domain is not bought yet

### Coderminal IDE
Coderminal IDE is a modern vibe-coding application which runs direclty on your PC.
- **Written In**: Electron
- **Status**: Developing
